# Estrutura TFS

- `src/`: motor C++.
- `data/`: conteudo do jogo.
- `data/scripts/`: RevScriptSys e sistemas modernos.
- `data/events/`: callbacks centrais, incluindo `Player:onNetworkMessage`.
- `data/lib/`: bibliotecas Lua.
- `schema.sql`: banco.
- `config.lua`: configuracao.
- `CMakeLists.txt`: build.
