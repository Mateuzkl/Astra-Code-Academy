Achados importantes

Os arquivos locais mostram uma migracao real de sistemas quentes para protocolo binario.

  • data/lib/core/packet_handler.lua define PacketHandler e NetworkGuard.
  • data/events/scripts/player.lua encaminha Player:onNetworkMessage para PacketHandlers[recvByte].
  • data/scripts/network/task_board/protocol.lua usa bytes reais para task board, soulseals e resource balance.
  • src/luanetworkmessage.cpp bloqueia opcodes Astra-only para clients que nao sao AstraClient.
  • forge/stress-test-results.html registra 31/31 testes passando, incluindo NetworkMessage 11/11.
